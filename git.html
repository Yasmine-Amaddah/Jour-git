Tableau des commandes Git

Commandes Description
git config Configurer les préférences de l’utilisateur
Git init créer un nouveau dépôt
Git add créer un nouveau dépôt
Git clone utilisée pour la vérification des dépôts
Git commit valider les modifications apportées au HEAD
Git status affiche la liste des fichiers modifiés ainsi que les fichiers qui doivent encore être ajoutés ou validés
Git push nvoie les modifications locales apportées à la branche principale associée
Git checkout créer des branches ou pour basculer entre elles
Git remote permet à un utilisateur de se connecter à un dépôt distant
